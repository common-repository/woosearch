WooSearch is a search plugin which provides real-time ajax product search based on WooCommerce.

== Description ==
WooSearch is a search plugin which helps your customer search products based on WooCommerce. You can setup promotion keywords and guide your customers to popular search results. WooSearch also provides real-time ajax search which gives better user experience to the customer.

= Real-time Ajax Search =
* **Free** built-in real-time ajax product search functionality

= Promotion Keywords =
* Pre-defined keywords can be set

== Installation ==

= Minimum Requirements =

* WooCommerce installed
* WordPress 4.0 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.1 or greater

= Manual installation =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'WooSearch'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. Add search box widget: Under WordPress admin page ==> Appearance ==> Widgets and drag WooSearch widget to your preferred location.
